# EgeCare
A responsive, serene landing page for a conceptual university wellness platform. Built with semantic HTML5 and modern CSS3 Grid/Flexbox.


🌿 EgeCare: University Wellness & Personal Care Platform
A modern, serene, and fully responsive landing page designed for "EgeCare," a conceptual wellness, mental health, and personal care initiative tailored for university students.

🚀 About the Project
EgeCare was built to provide a digital sanctuary for students dealing with campus life fatigue . The UI/UX design focuses on calmness and clarity, utilizing a minimalist aesthetic, ample white space, and a soothing earth-tone color palette . This project demonstrates solid foundational front-end skills, relying strictly on Vanilla HTML5 and CSS3 without the use of heavy frameworks.

✨ Key Features
🧘‍♀️ Wellness-Oriented UI/UX: Features a soft, comforting color scheme (#B2AC88, #F9F8F3) and an elegant typography pairing (Cormorant Garamond for headings, Montserrat for body text) to evoke a premium clinic or spa feel .

📱 Modern CSS Grid System: The "Services" and "Blog" sections are built using advanced CSS Grid (grid-template-columns: repeat(auto-fit, minmax(300px, 1fr))), ensuring fluid responsiveness across all mobile, tablet, and desktop screens .

🧭 Smooth Navigation: Implemented native CSS scroll-behavior: smooth for elegant, seamless transitions between page sections (Home, About, Services, Detail, Blog, Contact).

🎨 Interactive Elements: Includes subtle hover animations on buttons and cards (transform: translateY(-3px)) with carefully tuned box shadows to create depth and interactivity.

📌 Sticky Header: A persistent, glass-like navigation bar that stays accessible as users scroll through the platform .

🛠️ Technologies Used
Frontend Structure: Semantic HTML5 

Styling & Layout: CSS3 (Flexbox, Grid, CSS Variables) 

Typography: Google Fonts
